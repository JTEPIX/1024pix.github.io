Chaque ligne est décrite par une liste de nombres :
  • le premier donne un nombre de carrés blancs,
  • le deuxième un nombre de carrés noirs,
  • le troisième un nombre de carrés blancs,
  • et ainsi de suite...
Par exemple :
1, 2, 3, 1


En suivant la description de chaque ligne, reproduisez le dessin.
Il suffit de cliquer sur une case pour la faire changer de couleur.
